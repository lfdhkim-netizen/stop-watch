# /franchise-location-analyzer — 프랜차이즈 신규점 입지 분석

사용자가 이 슬래시 커맨드를 호출했다. **반드시** `franchise-location-analyzer` 스킬(`.claude/skills/franchise-location-analyzer/SKILL.md`)의 규칙을 따라 분석을 수행하라.

---

## 사용자 입력
사용자가 함께 입력한 인자: **$ARGUMENTS**

---

## 처리 순서

### Step 0. 입력 수집

사용자 인자에 **필수 항목**이 모두 들어 있는지 확인:

| 필수 | 항목 |
|---|---|
| ✅ | 후보 점포 주소 |
| ✅ | 상권 특성 (CBD / 오피스 외곽 / 혼합 / 주거 / 학원가 / 역세권) |
| ✅ | 임대조건 — 보증금, 월세, 권리금 |
| ✅ | 점포 면적 (평수, 좌석수) |
| 선택 | 반경 300m·500m·1km 직장인 인구 |
| 선택 | 인근 경쟁점 정보 (상호/단가/평점) |
| 선택 | 임대 계약기간·갱신조건 |
| 선택 | 지하철역 도보거리 |

부족하면 `AskUserQuestion` 도구로 **한 번에 묶어서** 질문하라. (4개 이하 질문으로 분할)
중복 질문 금지. 사용자가 이미 답한 항목은 다시 묻지 않는다.

### Step 1–6. 분석 실행

`SKILL.md`에 정의된 6단계를 그대로 수행:

1. 상권 분석 — 상권 분석가 페르소나
2. 타겟 적합도 — 직장인 고객 + 마케팅 전문가
3. 경쟁 분석 — F&B 운영 전문가
4. 재무 시뮬레이션 — 재무 분석가 (3시나리오)
5. 리스크 분석 — 부동산 감정평가사 + 가맹점주
6. 8인 페르소나 종합 평가 + GO/HOLD/NO-GO — 본부 의사결정자

### Step 7. Word 보고서 생성

`anthropic-skills:docx` 스킬을 호출하여 `.docx` 파일을 생성한다.

파일 저장 경로:
```
./reports/<YYYY-MM-DD>_<주소-약어>_location-analysis.docx
```

(예: `./reports/2026-05-14_강남-테헤란로_location-analysis.docx`)

### Step 8. 최종 출력

채팅 응답에는 다음만 간결하게 출력한다:

1. **판정**: 🟢 GO / 🟡 HOLD / 🔴 NO-GO 한 줄
2. **핵심 근거 3가지** (불릿)
3. **종합 점수표** (5개 영역 + 종합)
4. **조건부 GO/HOLD인 경우**: 충족 조건 (정량적)
5. **NO-GO인 경우**: 결정적 리스크 2–3개
6. **보고서 파일 경로**

상세 본문은 .docx에 담고, 채팅은 결재용 요약만 보여준다.

## 동작 규칙 (재확인)

- 모든 추정치는 출처·가정 명시
- 정량 점수 없는 판단 금지
- 사용자가 입력한 숫자는 그대로 사용 (변경 금지)
- 부족 정보는 추정 + "추정" 표시
- 판정 임계값: GO ≥ 75 평균 + 리스크 ≤ 30 / HOLD 60–74 / NO-GO < 60 또는 리스크 > 50
- 언어: 한국어, 본부 결재용 톤
