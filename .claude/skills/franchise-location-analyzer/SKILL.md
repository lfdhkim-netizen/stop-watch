# Franchise Location Analyzer — 프리미엄 국밥 프랜차이즈 입지 분석 스킬

## 무엇을 하나요
후보 점포의 **주소 + 상권 특성 + 임대조건**을 입력하면, 한 번의 호출로 다음 6단계를 자동 수행합니다.

1) **상권 분석** (도시계획·상권분석 컨설턴트 관점 — 유동인구/배후수요/동선/접근성)
2) **타겟 적합도 평가** (직장인 점심수요 × 15,000원 단가 수용성 매트릭스)
3) **경쟁 분석** (반경 500m 한식·국밥·점심특화 매장 매핑 + 가격/평점 벤치마킹)
4) **재무 시뮬레이션** (일·월 예상매출 / BEP / 투자회수기간 / 단위경제학)
5) **리스크 분석** (임대료·권리금·계약조건·시즌·환기/냉방·식자재 동선)
6) **8인 멀티 페르소나 평가 + GO/HOLD/NO-GO 판정** (최대 3회 리라이팅)

산출물은 **Word 보고서(.docx)** 한 묶음 — 요약 1페이지, 본문, 페르소나 평가 매트릭스, 최종 추천.

---

## 입력 (사용자가 직접 텍스트로 제공)

**필수**
- 후보 점포 주소 (도/시/구/동, 가능하면 지번 또는 도로명까지)
- 상권 특성 (예: CBD 오피스 / 오피스 외곽 / 오피스+주거 혼합 / 학원가 / 역세권 등)
- 임대조건: 보증금 / 월세 / 권리금
- 점포 면적 (평수, 좌석수)

**선택 (있으면 정밀도↑)**
- 반경 300m·500m·1km 추정 직장인 인구
- 인근 경쟁 한식·국밥점 정보 (상호/단가/평점)
- 임대 계약기간·갱신조건
- 인근 지하철역 도보거리

## 페르소나 (8명) — 각자 자기 관점에서 평가하고, 본부 의사결정자가 종합 판정

### 1. 상권 분석가 — 도시계획 박사, 15년차
- **역할**: 유동인구 추정, 동선 분석, 배후수요 측정, 접근성 평가
- **평가 기준**: 1차 상권(반경 500m) 유효 직장인 수, 도보 5분 내 동선 폭, 가로 활성도
- **언어**: 데이터·지표 위주, 정량적

### 2. 부동산 감정평가사 — F&B 임대차 전문
- **역할**: 임대료 적정성, 권리금 시세 비교, 입지가치 산정
- **평가 기준**: 평당 임대료 vs. 동급 상권, 권리금 회수 가능성, 계약조건 독성조항
- **언어**: 시세표·비교사례 인용

### 3. F&B 운영 전문가 — 15년차 외식 컨설턴트, 다수 한식 프랜차이즈 자문
- **역할**: 좌석수 적정성, 동선 효율, 회전율 시뮬레이션
- **평가 기준**: 점심 피크 90분 회전 가능 횟수, 주방 동선, 홀 서비스 효율
- **언어**: 운영 KPI 중심 (객단가·회전율·인건비율)

### 4. 재무 분석가 — CFO 출신, 외식 IPO 자문 경험
- **역할**: 단위경제학(unit economics), BEP, 투자회수기간
- **평가 기준**: 매출총이익률, 임대료/매출 비율, 4FL(식재·인건·임대·기타) 합계
- **언어**: 숫자 위주, 시나리오 3안(보수/기준/낙관)

### 5. 마케팅 전문가 — 외식 브랜드 마케터, SNS·배달앱 운영
- **역할**: 브랜드 노출 가능성, 배달앱 매출 기여, 직장인 SNS 도달
- **평가 기준**: 가시성(간판/외관), 인스타그래머블 요소, 배달 반경 내 오피스 밀도
- **언어**: 마케팅 funnel 관점

### 6. 가맹점주 — 다점포(3개 이상) 운영 경험, 현장형
- **역할**: 현장 운영 부담, 야간/주말 매출 현실성, 인력 수급
- **평가 기준**: 인근 알바 시급 시세, 직원 출퇴근 동선, 주말 상권 활성도
- **언어**: 현실적·체감적 ("이 동네는 토요일 죽어요" 같은 톤)

### 7. 직장인 고객 페르소나 — 30대 사무직, 점심값 1.2–1.5만원 허용, 회식 월 2–3회
- **역할**: 단가 수용성, 메뉴 매력도, 동료와 함께 갈 만한지
- **평가 기준**: "15,000원 점심 부담스러운가? 회식으로 적합한가? 법카 결제 가능한가?"
- **언어**: 1인칭 ("나라면 일주일에 한 번 정도는…")

### 8. 본부 의사결정자 — 개발본부장, 최종 GO/NO-GO 권한
- **역할**: 위 7인 평가를 종합 + 브랜드 포트폴리오 적합도 + 최종 판정
- **평가 기준**: 표준점포 모델 부합도, 본부 매뉴얼 적용 가능성, 리스크 허용한도
- **언어**: 경영진 의사결정 톤, 명확한 GO/HOLD/NO-GO + 조건부 사항

## 처리 단계 (한 번의 호출로 자동 수행)

### Step 1. 상권 분석 (상권 분석가 페르소나)
- 주소 기반 상권 유형 분류 (CBD / 오피스 외곽 / 혼합 / 주거 / 학원가 등)
- 1차 상권(500m) / 2차 상권(1km) 배후수요 추정
- 점심 유동인구 추정 (오피스 임직원 수 × 외식률 65–75% × 한식 선택률)
- 접근성: 지하철역 도보거리, 주변 주차, 주요 동선
- 산출: 상권 적합도 점수 (0–100)

### Step 2. 타겟 적합도 (직장인 고객 페르소나 + 마케팅 전문가)
- **15,000원 단가 수용성 매트릭스**
  - 인근 점심 평균단가 벤치마킹
  - 회사 식대 보조 평균(8,000–12,000원) 대비 자기부담분
  - 객단가 시뮬레이션: 사이드 (계란/공깃밥/김치추가) + 음료 → 17–18,000원
- **회식·저녁 매출 비중**
  - 주류 메뉴 가능성, 단체석 운영, 법인카드 결제 빈도
- **요일·시즌 편차**
  - 월·금 회식 피크, 여름철 국밥 수요 -20~30% 가능성
- 산출: 타겟 적합도 점수 (0–100)

### Step 3. 경쟁 분석 (F&B 운영 전문가)
- 반경 500m 내 한식·국밥·국물요리 매장 리스트
- 단가·평점·SNS 노출도 벤치마킹
- 차별화 포인트 (프리미엄 포지셔닝 vs. 기존 8,000–12,000원 국밥집)
- 카니발리제이션 리스크 (자사 직영점/가맹점 거리)
- 산출: 경쟁 우위 점수 (0–100)

### Step 4. 재무 시뮬레이션 (재무 분석가)
3가지 시나리오 (보수/기준/낙관) 산출
- **매출 추정**
  - 점심: 좌석수 × 회전율 × 점심 객단가 × 점심영업일
  - 저녁: 좌석수 × 회전율 × 저녁 객단가 × 저녁영업일
  - 배달: 점심 매출의 15–25%
- **비용 구조 (4FL)**
  - 식재료비 (30–35%)
  - 인건비 (25–28%)
  - 임대료 (월세/매출 8–12% 목표)
  - 기타 (광열·소모품·로열티 등 10–12%)
- **BEP / 투자회수기간** (보증금+권리금+인테리어+초기운영자금 기준)
- 산출: 재무 매력도 점수 (0–100)

### Step 5. 리스크 분석 (부동산 감정평가사 + 가맹점주)
- 임대료 압박 (월세/예상매출 비율)
- 권리금 회수 시나리오
- 계약 독성조항 (재계약·임대료 인상 한도)
- 환기/냉방 시설 (국물요리 특성)
- 식자재 신선도 동선 (새벽 도매 입고 가능성)
- 인력 수급 (인근 알바 시급, 야간 운영)
- 산출: 리스크 점수 (0–100, 낮을수록 좋음)

### Step 6. 8인 페르소나 평가 + 최종 판정 (본부 의사결정자)
- 페르소나 8인이 각자 1–2 문단 코멘트
- 평가 매트릭스: 5개 점수 + 종합점수
- **최종 판정**: 🟢 GO / 🟡 HOLD (조건부) / 🔴 NO-GO
- **조건부 GO인 경우**: 충족해야 할 조건 명시 (예: 권리금 협상 -20%, 환기시설 추가설치 등)
- **NO-GO인 경우**: 가장 결정적인 리스크 2–3개

## 사용 예시

**사용자 입력**
```
주소: 서울 강남구 테헤란로 ○○○
상권: CBD 오피스 (테헤란로 GBD 중심)
임대: 보증금 1.5억 / 월세 1,200만원 / 권리금 1.5억
면적: 35평 / 좌석 36석
임직원 추정: 반경 500m 약 25,000명
지하철: 역삼역 도보 4분
```

**스킬 출력**
- 6단계 분석 자동 수행
- 8인 페르소나 평가 매트릭스
- Word 보고서(.docx) 파일 (10–15페이지)
- 최종 판정 (예: 🟡 HOLD — 조건: 권리금 1.2억 이하 협상 + 점심 객단가 +α 메뉴 보강)

## 동작 규칙

1. **모든 추정치는 출처·가정을 명시한다** (예: "외식률 70% 가정 — 한국외식산업연구원 2024 보고 기준")
2. **숫자 없는 판단을 금지** — 모든 페르소나 평가는 점수(0–100) + 근거 1문장 포함
3. **GO 판정은 5개 영역 평균 75점 이상 + 리스크 영역 점수 30점 이하인 경우에만**
4. **HOLD는 충족 조건을 반드시 정량적으로 명시** (예: "권리금 -20%")
5. **사용자가 입력한 숫자는 변경하지 말고, 부족한 정보는 추정 + "추정" 표시**
6. **docx 생성 시 anthropic-skills:docx 스킬을 호출**, 본 스킬은 콘텐츠 생성에 집중
7. **언어**: 한국어. 보고서 톤은 본부 결재용(공식·간결·근거 중심)

## 자주 발생하는 케이스 대응

| 케이스 | 대응 |
|---|---|
| 주소만 있고 상권 유형 미입력 | 주소 기반으로 상권 유형 추정 후 "추정" 명시, 사용자 확인 요청 |
| 경쟁점 정보 누락 | 일반론적 카테고리 분석으로 대체, 정확도 한계 명시 |
| 임대조건 일부 누락 | 동급 상권 시세로 추정, 가정 박스에 명시 |
| 좌석수 미입력 | 평수 × 1.0–1.2석/평으로 추정 |
| 단가/메뉴가 다른 경우 | 입력된 단가·타겟으로 매트릭스 자동 재조정 |
